# 💡BookShare(독서 기록 서비스)
### 내가 읽은 책을 기록하며 다른 사람과 생각을 공유하는 서비스<br>



## 📝BookShare는 이런 분들을 위한 서비스입니다

- 평소 책을 즐겨 읽는 분
- 책을 기반으로 친목을 다지고 싶은 분
- 다른 사람의 생각이 궁금하신 분
- 책을 읽는 습관을 들이고 싶은 분
